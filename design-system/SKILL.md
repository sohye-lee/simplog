If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.

Ledger in one line: monotone warm-neutral ink + one point color (Ledger Lime `#CCF017`), IBM Plex Mono type with tabular figures, small crisp corners, borders over shadows, no emoji, no imagery. Amounts always lead; the lime is loud — one per view.
