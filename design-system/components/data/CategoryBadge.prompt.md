Compact category chip. Categories are distinguished by label, not color (Ledger is monotone). `selected` fills it lime — use for the active filter.

```jsx
<CategoryBadge>Food</CategoryBadge>
<CategoryBadge selected>Rent</CategoryBadge>
<CategoryBadge dot={false} size="sm">Transport</CategoryBadge>
```

Props: `selected`, `dot`, `size` (`sm`|`md`).
