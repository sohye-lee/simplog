Styled native select for category and month pickers.

```jsx
<Select label="Category" placeholder="Choose…"
  options={['Food', 'Rent', 'Transport', 'Fun']} />
<Select options={[{value:'2026-07', label:'July 2026'}]} />
```

Props: `label`, `options` (strings or `{value,label}`), `placeholder`, `invalid`.
